# Iris_DataSet_Visualization
Iris Dataset Exploratory Data Analysis (EDA) project using Pandas, Matplotlib, and Seaborn for data inspection and visualization.

## Task Objective

The objective of this project is to learn how to load, inspect, and visualize a dataset to understand its structure, trends, and value distributions using Python data analysis libraries.

## Dataset Used

* **Dataset:** Iris Dataset
* **Source:** CSV file loaded using Pandas
* **Description:** The Iris dataset contains 150 samples of iris flowers from three species: Setosa, Versicolor, and Virginica. It includes four numerical features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width

## Models Applied

No machine learning models were applied in this task. The focus was on Exploratory Data Analysis (EDA) using:

* Pandas for data loading and inspection
* Matplotlib for plotting
* Seaborn for enhanced visualizations

## Key Results and Findings

* Successfully loaded and inspected the Iris dataset using Pandas.
* Used `.shape`, `.head()`, `.info()`, and `.describe()` to understand the dataset structure and summary statistics.
* Scatter plots revealed relationships between flower features and species.
* Histograms showed the distribution of numerical values across features.
* Box plots helped identify data spread and potential outliers.
* The analysis provided insights into the characteristics and patterns present in the Iris dataset.
